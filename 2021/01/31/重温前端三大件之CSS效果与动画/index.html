CSS 动画

动画的原理：

  • 视觉暂留作用
  • 画面逐渐变化

动画的作用

  • 愉悦感
  • 引起注意
  • 反馈
  • 掩饰（加载过程）

CSS 中的动画类型

面试题：CSS 中实现动画有多少种方案，transition 型过渡动画，一种是 animation（keyframes）关键帧动画

  • transition 补间动画
  • keyframe 关键帧动画
  • 逐帧动画

补间动画

  • 位置-平移 （left / right / margin / transform）
  • 方位-旋转（transform）
  • 大小-缩放（transform）
  • 透明度（opacity）
  • 其它-线性变换 （transform）

自定义动画

关键帧动画

  • 相当于多个补间动画

  • 与元素状态的变化无关

  • 定义更加灵活（进度可控等）

逐帧动画

  • 适用于无法补间计算的动画
  • 资源较大
  • 使用 steps()

比如说8张连续图放八段显示

CSS 面试真题

CSS 动画的实现方式有几种

  • transition
  • keyframes（animation）

过渡动画和关键帧动画的区别

  • 过渡动画需要有状态变化
  • 关键帧动画不需要状态变化
  • 关键帧动画能控制更精细

如何实现逐帧动画

  • 使用关键帧动画
  • 去掉补间（ steps）

CSS 动画的性能

  • 性能不坏
  • 部分情况下优于 JS
  • 但 JS 可以做到更好
  • 部分高危属性 box- shadow 等
